# ConsumerLens

ConsumerLens 是一款 **AI 驱动的消费品市场调研工具**，面向需要在短时间内理解陌生品类、并快速完成 **secondary research（案头研究/二手资料调研）** 的消费品产品经理（PM）。它将信息采集、结构化整理、竞品与价格带分析、结论产出整合到同一工作流中，帮助你更快形成可行动的调研结论。

## 项目简介

在做新品/新类目判断、渠道定价与供需验证、竞品对标时，PM 往往需要在多个平台反复搜索、整理、对比与归纳。ConsumerLens 以“问题驱动 + 结构化输出”为核心，把分散信息转化为可复用的研究材料（要点、表格、结论、建议），显著降低调研时间成本。

## 核心功能

- **陌生品类速览**：用统一框架输出品类定义、关键人群、核心需求、关键参数与购买决策因素。
- **Secondary research（案头研究）**：聚合公开资料与线上信号（电商、社媒、社区、媒体商业报告等），抽取价格带与主流规格、典型卖点/槽点、高频需求与使用场景。
- **竞品与价格带分析**：对比品牌/型号/版本差异，归纳卖点、槽点、典型使用场景与价格带结构。
- **信息结构化与证据链**：把原始材料转为结构化卡片/要点，支持保留来源与关键引用，方便复核与追溯。
- **结论与建议生成**：输出可直接用于汇报的摘要、洞察、机会点与下一步调研建议（可按模板生成）。
